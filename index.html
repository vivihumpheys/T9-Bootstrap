Sobre

Profissional multilíngue e transdisciplinar, Viviane é carioca, atriz, formada em Comunicação Social com ênfase em Audiovisual e pós em Gestão de Negócios.

Atualmente, está em fase de transição de carreira para a área de tecnologia, com foco em programação front-end.

Além da área de front-end, tem interesse em aprender mais sobre cyber-security, data-science e scrum.

Portfólio

M de Maravilhosas

Escolhemos homenagear mulheres inspiradoras. No meu projeto falei de Frida Kahlo, artista e pintora mexicana. Projeto realizado com HTML e CSS.

Sobre o projeto

Esse projeto foi realizado no primeiro módulo do bootcamp de front-end da @reprograma. As alunas criaram uma página com uma breve biografia de uma mulher que as inspira. Utilizamos HTML e CSS para realizar o projeto.

Lista de Tarefas

To-do List, criado com JavaScript, HTML e CSS, com funcionalidade de adicionar, excluir e marcar tarefas realizadas e reorganizá-las com 'drag and drop'.

Sobre o projeto

Essa aplicação foi criada como conclusão do segundo módulo do bootcamp de front-end da @reprograma. As alunas criaram o código em JavaScript para realizar uma lista de tarefas. Também foi utilizado HTML e CSS.

Miranday

Miranday é um projeto pessoal em formato de vídeos na plataforma YouTube. O projeto trás esquetes de humor e observações da autora sobre a vida.

Sobre o projeto

Esse projeto foi idealizado em 2018 e foi veiculado inicialmente na plataforma IGTV, do Instagram. Depois foi lançado também como canal de YouTube. Os vídeos trazem esquetes escritas, dirigidas e interpretadas por Viviane Miranda, com temas cotidianos e tons de humor.

Biscoito da Sorte

Aplicação criada com JavaScript no módulo 2 do bootcamp de front-end da @reprograama.

Sobre o projeto

Essa aplicação foi criada como exercício do módulo 2 do curso. Criamos um biscoito da sorte virtual, em que cada clique é oferecida uma frase de sorte ao usuário. Foram utilizados JavaScript, HTML e CSS.
